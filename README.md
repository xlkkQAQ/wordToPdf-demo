# docxToPdf-demo

## 简介

该项目是一个word转pdf的小demo，该项目使用了两种方式集成，分别是**docx4j**和**aspose**



## 结果展示

docx4j对格式有一定限制，导入word文档大概率格式会乱，就不展示了。

### aspose

运行项目，默认进入该界面（过于简陋，后续优化），上传本地word文件，点击转pdf按钮，会自动转pdf并展示。用户可以直接下载下来。
